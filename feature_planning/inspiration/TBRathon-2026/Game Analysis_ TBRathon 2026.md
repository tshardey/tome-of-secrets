# **Game Analysis: TBRathon 2026**

## **1\. Global System Parameters**

These rules apply to all modules within the TBRathon 2026 system.

* **Timelines:**  
  * **Weekend-long:** First week of January 2026\.  
  * **Month-long:** Entire month of January 2026\.  
  * **Year-long:** Entirety of 2026 (events every 1st weekend of the month).  
* **Flexibility Settings:**  
  * **Stacking:** ALLOWED. One book can fulfill multiple prompts across different challenges.  
  * **Genre Blending:** ALLOWED. (e.g., A Fantasy-Romance counts for both Fantasy and Romance prompts).  
  * **Formats:** Short stories, audiobooks, and graphic novels are valid unless specified otherwise.  
  * **Completion:** DNFs (Did Not Finish) count as "read" if the user chooses.  
  * **Order:** Prompts can be completed in any sequence.

## **2\. Game Modules**

The system consists of three distinct "sub-games." Users may play one, two, or all three.

### **Module A: TBR Explorer**

Concept: A "Build Your Own Adventure" model.  
Core Mechanic: The user must collect exactly 12 qualities total.  
Constraint: Qualities can be chosen from any of the 5 categories (Genre, Setting, Main Character, Review, Plot) in any distribution.

* *Example:* User chooses 1 Genre, 5 Settings, and 6 Plot points \= 12 total.

### **Module B: Lost in the Library**

Concept: A board game traversal simulation.  
Core Mechanic: Dice rolling or random selection to move between Rooms and Hallways.  
Goal: Visit rooms or squares to collect prompts.  
Input Method: Users can ask the Agent to "Roll a D6" to determine their next prompt, or manually select.

### **Module C: Map of Stories**

Concept: Geographic traversal of genre-themed locations.  
Core Mechanic: Visit 12 locations. Each location has 3 specific prompts.  
Difficulty Modes:

* **Easy Mode:** Complete prompts regardless of genre.  
* **Hard Mode:** Books read must match the location's specific *Genre*.

**Badge System Logic:**

* **Gold Badge:** Complete ALL 3 prompts at a location \+ Match Genre.  
* **Silver Badge:** Complete ALL 3 prompts at a location (Genre ignored).  
* **Rose Badge:** Complete 1 prompt at a location (Genre ignored).
